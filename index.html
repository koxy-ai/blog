What is Koxy AI for real ?

August 21, 2023 Â· 3 min read
Kais Radwan
Founder & developer at Koxy AI

Koxy AI is more than just a no-code platform for building a cloud-based API, The system capabilities goes beyond that and We will discuss in this post what you can build with Koxy AI. let's go..!

Using Koxy AI the bad way ?â

A lot of people around the social media have been talking about how Koxy AI can be misused or used to do bad things, and how the idea is being misunderstaned by a lot of people, the fact that Koxy AI is a powerful system is true, and We are currently doing our best to restrict all the ways that can be used to do bad things throught Koxy AI edge-cloud.

We are taking all the concerns in a very serious way and doing our best to make the technology safer everyday.

Koxy AI's core is the KSS (Koxy Simulation System) that does all the magic. You just build a simple JSON object and It turns it into an API, so basically You give it the API logic and It builds the actual code and run it in the cloud, served from 35 regions around the world at the edge. and at the edge means from the closest region to your users.

And yes, Koxy AI can serve your API at the speed of 2ms to 10ms if you are not doing any heavy job. and It can iterate on a very high speed and It's great to do DDoS attacks and We are currently working our best to build a safty system to block any pattern similar to any of these attacks, believe me no one wants that to be available to anyone for only $12.

Look at the numbers below:

We've served this huge number of requests in a week and our system is ready to serve millions more without any issues.

How is Koxy AI designed ?â

Well, if you're talking about the platform It's a really simple React website, the big thing is the API builder, All your API endpoints and nodes are just JSON objects that can be parsed by the simulation system.

If You're talking about the edge-cloud, It's where the simulation system hosted, It's built on top of Deno and uses GUN DB for real-time decentralized communications.

And about the simulation system, We are going to publish a full document about how It's built and how It works soon.

The stable version is here

August 15, 2023 · 3 min read
Koxy Team
Koxy AI blogging team

We are releasing the stable version of Koxy AI. Let's explore the main new features, updates, and fixes in the new release.

Platform's UI/UX​

Well, this might be the most important feature in this update, We've upgraded the user experience to a whole new level to meet our users expectations from a cutting-edge product like Koxy AI... the theme, colors, designs, and layout has been updated.

Now you'll never be lost... and everything will be clear for you so you get the best out of Koxy AI.

the new version is now live

API Builder​

Big things coming to the API builder. first let's take a look on the old and new builder:

old builder:

new builder:

New Features:

  • Console: Run your API in the console with new UI design and UX experience.

  • Errors: Now the system will highlight the node where the error happened.

  • Run region: Now you'll be able to see in what region the run has been executed.

  • Builder actions panel: a panel at the bottom of the builder and it gives you the ability to run your API, send parameters, and add new nodes to your API.

  • Nodes connectors: Now you can connect nodes together from the Builder canvas or space using connectors.

Free plan​

Now there is a free plan available to everyone of you. you get 50 cloud runs to try Koxy AI and explore its powers. get your own right now.

Cloudspace Setup​

We've made it much easier for you to setup your Cloudspace and payment and start building your API.

Payments​

There was some bugs during the payments process, We fixed all the bugs and issues and made the process much faster and easier.

Settings​

We added new configurations to the settings like the auto_run feature (to run your API automatically everytime you make a new deployment).

Logs​

This feature has been released earlier, but it deserves to talk about it here. you can keep tracking of your logs and see the logs of every run in the console.

Analytics​

This feature has been released earlier too. Now you can track the number of requests to your API / month.

Profile​

We updated the profile's UI too. this is how it looks like now:

Errors​

Now you'll get a better error handling features and more clear errors, with smart highlighting to highlight the node where the error happened.

Warnings​

There was a confusing bug in the warnings where some warnings were logged twice, this bug is fixed now with more clear warnings.

Thank you​

We need to thank all of you for your support. We released the experiment version of Koxy AI just a week ago and We got a lot of support from the community, a lot of developers already started building cool projects using Koxy AI, a lot are still testing it, a lot didn't get a chance to try the free version... thank you all, You are all helping us make Koxy AI better.

What's next​

We still have a lot of things to work on, for example the documentation now is out of date, We need to add languages to the platform, some users still experiencing latency in the edge-cloud, and a lot of new nodes are under development... We will continue working on making Koxy AI better for you everyday .!!
